# Carnet des sorties scolaires — École Jean Aicard

Application de suivi des sorties scolaires (2026–2027) : autorisations, DEPL,
cantine, commande de bus, agenda.

## Utilisation

Ouvrir simplement la page publiée par GitHub Pages. Les données sont
enregistrées automatiquement dans le navigateur utilisé (voir le bouton
« Exporter » pour sauvegarder ou transférer les données vers un autre
appareil).

## Mise à jour du contenu

Ce dépôt contient un seul fichier `index.html` autonome (HTML + CSS + JS,
aucune dépendance externe hormis une police Google Fonts). Toute modification
se fait en éditant ce fichier directement, y compris depuis l'éditeur en
ligne de GitHub.
